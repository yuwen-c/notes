# center in flexbox
- horizontal
```
justify-content: center
```
- vertical
```
align-items: center
```


## 把排成直列的items垂直置中
```
flex-direction: column // 轉成直的
```
- 因為已經轉過方向了，要用justify-content
```
justify-content: center
```
