# 選擇器會影響比重

```css
/* API欄位分上下兩格 */
// 這種寫法的比重會比下面那個寫法的比重高
  .udi-visibility + .apiContent {
    padding: 0;
  }

  .apiContent {
    padding: 0;
  }
```
