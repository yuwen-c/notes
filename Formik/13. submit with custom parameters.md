# default submit parameters
- values and formikBag
- FormikBag大禮包，可以在裡面拿到所有helper methods
- 也可以另外傳自己的參數進去

```jsx
const handleSubmit = (values, { resetForm }, qaNo) => {
    console.log('submit');
    const { context } = values;
    const params = {
      qaNo,
      message: context,
    };
    fetchReply(params).then(() => {
      const p = { orderQaNo: qaNo };
      fetchOneQA(p);
      resetForm();
    });
  };
```
