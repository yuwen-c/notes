# install eslint as a package
- 安裝： `npm install --save-dev eslint`
- 會需要回答一系列問題、頗容易

## Edit `.eslintrc.json` file created in *step 2*

- Add inside `extends` rule `"airbnb"`
- my current extends rule inside .eslintrc.json
